# sketch

Sketch is a tool for converting presets exported from
[Squiglink](https://squig.link) sites and adding them to the
[Neutron Music Player](https://neutroncode.com/apps/player) app. The
[generic](https://neutroncode.com/downloads/category/1-generic-android) variant
of the app saves user data in the `NeutronMP` directory found under `/sdcard`.
EQ presets are saved under _Frequency Response Correction_ and the data for this
section is saved in `fc_presets.xml`.
